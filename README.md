# 64ます計算 PDFジェネレーター

子供の学習用に、A4サイズで印刷できる「64ます計算」のPDFを自動生成するPythonスクリプトです。
指定された形式（2桁と1桁の足し算・引き算の2つのグリッドが1ページに収まり、生徒用と教師用解答が交互に出力される形式）を完全に再現しています。

## 特徴
- 毎回ランダムな数値で問題を生成
- 1ページに「足し算」と「引き算」の2つの64ます計算を配置
- 生徒用ページと教師用（解答付き）ページが交互に生成される（デフォルトで計16ページ）
- 上部の数値は1桁（2〜9）、左側の数値は2桁（11〜19）で固定
- 実際のプリントと同じ明朝体フォント（HeiseiMin-W3）とレイアウトを採用

## インストール

```bash
pip install -r requirements.txt
```

## 使い方

基本の実行（8セット、計16ページのPDFを作成）:
```bash
python generate_pdf.py
```

セット数を変更する場合（例: 3セット作成）:
```bash
python generate_pdf.py -n 3
```
